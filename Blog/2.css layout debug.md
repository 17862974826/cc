## 二、源码解读

### 1、选取页面所有DOM元素
函数`$$()`;在现代浏览器的API中几乎都有支持，等价于 `document.querySelectorAll();`

$$('*')将所有DOM元素转成NodeList对象，但这并不是一个JS数组，所以不能用`$$('*').forEach()`方法来迭代，所以使用call或者apply方法来使用foreach



```
function(a){                             // 
  [].forEach.call(                       // 
                                         // 
    document.querySelectorAll(a),        // 
                                         // 
    function(b){                  
      b.style.outline = "1px solid #" +  //  
      (~~(Math.random()*(1<<24)))        // 
      .toString(16)})                    // 
}
```
