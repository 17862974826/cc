### 2.1 数据类型
* 原始数据类型
	* number
	* string
	* boolean
* 合成数据类型
	* object
	* array
	* function
* null undefined
	* undefined == null //true
	* Number(null) // 0
	* Number(undefined) // NaN
	* null 表示“没有对象” 此处不应该有值
	* undefined 表示“缺少值” 即此处应该有一个值，但还未定义
* typeof
	* typeof [] // "object"
	* typeof null // "object"
	* [] instanceof Array // true
	* {} instanceof Arry // false
* if([])  if({})  // true
